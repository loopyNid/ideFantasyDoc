Manual of Magnetic Dance glove

General Overview: The glove sets up a wifi network. The computer running supercollider connects to this network and recieves the sensor data from the glove. In parallel it connects to a second wifi network which provides internet access.

Steps:

  • connect the computer running supercollider to the “glove network”
  • access internet via the glove

Connect to “glove network”

Firstly connect the glove to power. After 3-4 minutes your computer should de- tect a wifi network named MD. Connect to this wifi network using the password ChangeMe. After connection visit 10.3.141.1, there you will be asked for a username and password, use admin as the username and secret as the password. At this point you should see a dashboard like this.

Next we need to set the computers ip.

you may need to reconnect to MD for the ip to take effect. After this supercollider should recieve sensor data.

Access internet

Navigate to the “Configure WiFi client” tab at 10.3.141.1 in the browser. Here choose the network you want to connect to. Add the password and click on connect. Now reboot the glove by navigating to the “System” tab on the dashboard and clicking “Reboot”. After this you should have internet connection while recieving sensor data if connected to MD.
